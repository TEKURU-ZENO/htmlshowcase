Using PX Units

This section uses fixed pixel units for all measurements. Notice how the sizes stay fixed regardless of root font size changes.

When viewing on smaller screens or when users change their browser font settings, these pixel-based measurements won't adapt as well.

PX Button

Using REM Units

This section uses rem units which are relative to the root font size. Notice how everything scales proportionally when the root font size changes.

This makes the design more accessible and flexible across different devices and user preferences.
