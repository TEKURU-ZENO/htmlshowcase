CSS Positioning Power

CSS Position Property Examples

The CSS position property is used to specify how an element is positioned in a document. This page demonstrates the three most common positioning methods: fixed, relative, and absolute.

Relative Positioning

With position: relative, an element is positioned relative to its normal position. Setting the top, right, bottom, and left properties will move the element from its normal position.

position: relative
This box is positioned relatively.

Absolute Positioning

With position: absolute, an element is positioned relative to its nearest positioned ancestor (an element with position other than static). If no positioned ancestors exist, it uses the document body.

Container: position: relative
This box is positioned absolutely.

Fixed Positioning

With position: fixed, an element is positioned relative to the viewport, which means it always stays in the same place even if the page is scrolled.

Stacked Elements with z-index

The z-index property specifies the stack order of an element. An element with greater stack order is always in front of an element with a lower stack order.
